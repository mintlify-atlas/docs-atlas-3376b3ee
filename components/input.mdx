## Installation

<Tabs>
  <Tab title="CLI">
    ```bash
    npx shadcn-svelte@next add input
    ```
  </Tab>
  <Tab title="Manual">
    <Steps>
      <Step title="Copy component code">
        Copy and paste the component source code into your project.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Usage

```svelte
<script lang="ts">
  import { Input } from "$lib/components/ui/input/index.js";
</script>
```

```svelte
<Input />
```
